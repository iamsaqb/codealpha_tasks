# Student CGPA Management System

A console-based Student CGPA Management System built using C++.

## Features
- Add new student
- Enter course grades and credit hours
- Automatic CGPA calculation
- Performance classification
- File handling (saves report to student_report.txt)
- Input validation and exception handling

## Concepts Used
- Object-Oriented Programming (Classes & Objects)
- Vectors (STL)
- File Handling
- Exception Handling
- Input Validation
- Formatting with iomanip

## How to Run

Compile:
g++ main.cpp -o cgpa

Run:
./cgpa
