# Bank Management System

A simple console-based Bank Management System written in C++.

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Display Account Details

## Technologies Used
- C++
- File Handling (if used)
- Object-Oriented Programming

## How to Run

1. Compile:
   g++ main.cpp -o bank

2. Run:
   ./bank
